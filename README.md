# Joe's Hangout

A complete production-ready Next.js app for Joe's Hangout.

## Getting Started

### Prerequisites
- Node.js 14.x or later
- npm or Yarn

### Installation

1. Clone the repository:
   
   ```bash
   git clone https://github.com/joeshangout/joes_hangout.git
   cd joes_hangout
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

### Running the App

To run the app in development mode:

```bash
npm run dev
# or
yarn dev
```

### Building for Production

To create an optimized build:

```bash
npm run build
# or
yarn build
```

### Start the Production Server

To start the optimized production server:

```bash
npm start
# or
yarn start
```

### Deployment

1. Set up a hosting platform (e.g., Vercel, Netlify).
2. Connect with your GitHub repository and deploy it.
